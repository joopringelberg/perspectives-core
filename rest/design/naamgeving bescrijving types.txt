TERMINOLOGIE EN NAAMGEVING VOOR BESCHRIJVING VAN TYPES IN TERMEN VAN CONTEXTEN EN ROLLEN

HOE BESCHRIJVEN WE CONTEXTEN, ROLLEN EN PROPERTIES?
Als we "een context beschrijven", stellen we dus een context-type op. Oftewel: het resultaat van beschrijven is een type. Voor de diverse Perspect typen gebruiken we echter niet altijd hetzelfde Perspect type om het mee te beschrijven
We beschrijven alles met een Context, behalve:

	:RolInContextType, daarvoor gebruiken we namelijk :RolInContext
	:BuitenRolType, daarvoor gebruiken we namelijk :BuitenRol
	:BinnenRolType, daarvoor gebruiken we namelijk :BinnenRol
	:PropertyType, daarvoor gebruiken we namelijk :RolInContext


BESCHRIJVING VAN EEN CONTEXT
Een context wordt dus met een context beschreven. Zo'n context heeft als pspType ":ContextType".
Merk op dat pspType namen gewoon de naam van het Perspect concept is, waarachter we "Type" schrijven (en dan in de perspect namespace, die ik hier als default aanneem).

Zijn rdf:type zou :Context zijn, maar dat representeren we niet expliciet, omdat we twee typen datastructuren hebben, namelijk voor Rol en Context. Dat zou corresponderen met twee rdf:typen, namlijk :context en :rol (eigenlijk :binnenRol, :buitenRol en :rolInContext). We kunnen aan de Purescript vorm daarvan al zien dat het gaat om een Context of een Rol. Alledrie de soorten rollen representeren met hetzelfde purescript type PerspectRol.

De binnenrol (en buitenrol) van het beschreven type beschrijven we ook weer met een context. Deze context heeft als type :BinnenRolType (resp. :BuitenRolType).

Zo'n :BuitenRolType context wordt verbonden met de :ContextType context middels een rol, namelijk: :BuitenRol.

LET OP: de :ContextType context heeft zelf óók een buitenrol-property. Maar die heeft als waarde, hoe kan het anders, een instantie van :Buitenrol en die gebruiken we niet om het :BuitenRolType te beschrijven!

In schema:

	{ id: ":Aangifte"
	, pspType: ":ContextType"
	, binnenRol: {...}
	, buitenRol: ":Aangifte_BuitenRol"
	, rollen: 
		{ ":BuitenRol": ":BuitenRolTypeVanAangifte" 
		, ":BinnenRol": ":BinnenRolTypeVanAangifte"
		}
	}
	
	{ id: ":BuitenRolTypeVanAangifte"
	, pspType: ":BuitenRolType"
	, binnenRol: {...}
	, buitenRol: "BuitenRolBijAangifte_BuitenRol"
	, rollen:
		{ ":betrouwbaarheid": ":Aangifte_betrouwbaarheid"
		}
	}
	
	{ id: ":Aangifte_betrouwbaarheid"
	, 
	}

NAAMGEVING: de naam van de context die het type van de buitenrol van een context-type beschrijft, stellen we zo samen:

	BuitenRolTypeVan<context-type-naam>
