PURESCRIPT BINARY DOWNLOADEN
Ik heb de purescript binary gedownload van: https://github.com/purescript/purescript/releases.

Vervolgens moet ik die in een map plaatsen die in mijn PATH zit. 
Met 
	echo $PATH
zie ik wat mijn path is (onder de zojuist geinstalleerde node versie). Ik kies daaruit:
	/Users/joopringelberg/.nvm/versions/node/v6.14.2/bin
In die directory bleek al een symbolische link te zijn gemaakt naar de node installatie van purs, maar die was dus mislukt. Ik heb die link verwijderd en vervolgens de purs executable daar neergezet.
